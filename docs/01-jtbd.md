# JTBD (Jobs To Be Done)

## プロダクト名: グロースパック for LINE - スタンプカード機能

## メインジョブ

When **店舗で買い物をして会計をする時**
I want to **簡単にスタンプを貯めて、自分の会員ランクや特典を確認したい**
So that **お得に買い物ができ、次回の来店モチベーションが上がる**

## 関連ジョブ

### ジョブ1: リピート促進（店舗側）

When **顧客データを活用してリピート率を上げたい時**
I want to **スタンプ状況に応じたセグメント配信をしたい**
So that **休眠顧客の掘り起こしやロイヤル顧客の囲い込みができる**

### ジョブ2: ランクアップ達成（顧客側）

When **あと少しで次のランクに上がれそうな時**
I want to **自分の現在のステータスと必要なポイント数を確認したい**
So that **達成感を得て、さらに購買意欲が高まる**

### ジョブ3: ブランド体験（店舗側）

When **LINE公式アカウントでスタンプカードを提供したい時**
I want to **自社ブランドの世界観を反映したデザインにしたい**
So that **顧客に一貫したブランド体験を提供できる**

### ジョブ4: 顧客分析（店舗側）

When **マーケティング施策を検討する時**
I want to **誰がどれくらいスタンプを貯めているか把握したい**
So that **データに基づいた効果的なキャンペーンを実施できる**

## ジョブの優先順位

| 優先度 | ジョブ | 理由 |
|--------|--------|------|
| 1 | スタンプを貯める・確認する（顧客側） | コア機能。これがないとサービスが成立しない |
| 2 | ランクアップ達成 | 顧客のエンゲージメントを高める重要な動機付け |
| 3 | 顧客分析 | 店舗側の価値提案の核心部分 |
| 4 | セグメント配信 | データ活用の具体的なアウトプット |
| 5 | ブランドカスタマイズ | 差別化要因だが、MVPでは優先度を下げられる |
