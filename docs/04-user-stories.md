# ユーザーストーリーマッピング

## プロダクト名: グロースパック for LINE - スタンプカード機能

---

## ユーザーペルソナ

### ペルソナ1: エンドユーザー（消費者）

| 項目 | 内容 |
|------|------|
| 名前 | 田中 美咲（32歳） |
| 属性 | アパレルショップの常連客、LINEヘビーユーザー |
| 目標 | お気に入りのショップでお得に買い物をしたい |
| 課題 | 紙のスタンプカードをよく忘れる、特典の有効期限を把握できない |

### ペルソナ2: 店舗スタッフ

| 項目 | 内容 |
|------|------|
| 名前 | 佐藤 健太（28歳） |
| 属性 | アパレルショップ店長、顧客対応・販促担当 |
| 目標 | 顧客満足度を上げながら効率的にスタンプを付与したい |
| 課題 | 紙のスタンプカード管理が煩雑、顧客の特典状況が分からない |

## ユーザーアクティビティ（大きな流れ）

```

### 店舗スタッフの体験フロー

```

## ストーリーマップ

### アクティビティ1: スタンプカード登録

| タスク | ユーザーストーリー | 優先度 |
|--------|-------------------|--------|
| LINEミニアプリを開く | As a 顧客, I want to LINEから簡単にスタンプカードにアクセスしたい, So that 新しいアプリをインストールする手間が省ける | MVP |
| 会員登録する | As a 顧客, I want to LINE認証で簡単に登録したい, So that 入力の手間なく始められる | MVP |
| 初回特典を受け取る | As a 顧客, I want to 登録したらすぐに特典をもらいたい, So that 登録するモチベーションが上がる | 後回し |

### アクティビティ2: 来店・スタンプ獲得

| タスク | ユーザーストーリー | 優先度 |
|--------|-------------------|--------|
| QRコードを提示する | As a 顧客, I want to 会計時にQRコードを見せるだけでスタンプを貯めたい, So that スムーズに会計できる | MVP |
| スタンプ付与を受ける | As a 顧客, I want to 購入金額に応じたスタンプをもらいたい, So that たくさん買うほどお得になる | MVP |
| 付与完了を確認する | As a 顧客, I want to スタンプが付与されたことをすぐに確認したい, So that 安心できる | MVP |

### アクティビティ3: スタンプ確認・ランク確認

| タスク | ユーザーストーリー | 優先度 |
|--------|-------------------|--------|
| 現在のスタンプ数を見る | As a 顧客, I want to 今何個スタンプが貯まっているか確認したい, So that 次の特典までの道のりが分かる | MVP |
| ランクを確認する | As a 顧客, I want to 現在の会員ランクと特典内容を確認したい, So that 自分のステータスを把握できる | MVP |
| ランクアップ条件を見る | As a 顧客, I want to 次のランクまでの必要スタンプ数を知りたい, So that 目標を持って買い物できる | MVP |
| 履歴を確認する | As a 顧客, I want to スタンプの獲得履歴を見たい, So that 過去の買い物を振り返れる | 後回し |

### アクティビティ4: 特典利用

| タスク | ユーザーストーリー | 優先度 |
|--------|-------------------|--------|
| 利用可能特典を確認する | As a 顧客, I want to 今使える特典を一覧で見たい, So that お得に買い物できる | MVP |
| 特典を使用する | As a 顧客, I want to 会計時に特典を簡単に使いたい, So that 割引を逃さない | MVP |
| 特典の有効期限を確認する | As a 顧客, I want to 特典の有効期限を確認したい, So that 失効前に使える | 後回し |

## MVPスコープ

### MVP機能一覧

1. **スタンプカードホーム画面** - 現在のスタンプ数、ランク、次のランクまでの進捗を表示
2. **スタンプ獲得演出** - スタンプ付与時のアニメーション表示
3. **ランク制度表示** - 各ランクの特典内容と昇格条件を表示
4. **QRコード表示** - 店舗スタッフがスキャンするための会員QRコード
5. **特典一覧** - 利用可能な特典を表示

### 後回し機能

1. **スタンプ獲得履歴** - 過去のスタンプ獲得履歴詳細
2. **有効期限通知** - 特典期限切れ前のプッシュ通知
3. **友達紹介機能** - 紹介でボーナススタンプ
4. **ゲーミフィケーション** - バッジ・実績機能
5. **店舗検索** - 近くの対象店舗を表示
