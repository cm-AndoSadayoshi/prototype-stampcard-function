# 狩野モデル（機能優先順位）

## プロダクト名: グロースパック for LINE - スタンプカード機能

---

## 品質分類

### 当たり前品質 (Must-be) - 必須

ないと不満だが、あっても満足度は上がらない機能。

| 機能 | 説明 | MVP判断 |
|------|------|---------|
| スタンプ数表示 | 現在の累計スタンプ数を表示 | ✅ 必須 |
| スタンプ獲得機能 | QR読み取りでスタンプを付与 | ✅ 必須 |
| 会員QRコード表示 | 店舗スタッフがスキャンする会員識別用QR | ✅ 必須 |
| ランク表示 | 現在の会員ランクを表示 | ✅ 必須 |
| 特典一覧 | 利用可能な特典を一覧表示 | ✅ 必須 |

---

### 一元的品質 (Performance) - 選択的

あればあるほど満足度が上がる機能。

| 機能 | 説明 | MVP判断 |
|------|------|---------|
| スタンプ獲得履歴 | 過去の獲得履歴を詳細表示 | ⚪ 後回し |
| 複数ランク対応 | 3段階以上のランク設定 | ⚪ MVPは2〜3段階 |
| 有効期限表示 | 特典の有効期限を明示 | ⚪ 検討 |
| 店舗情報表示 | 対象店舗の情報・地図 | ⚪ 後回し |
| プッシュ通知 | ランクアップ・期限切れ前通知 | ⚪ 後回し |

---

### 魅力的品質 (Delighter) - 1つに集中

なくても不満はないが、あると感動する機能。

| 機能 | 説明 | MVP判断 |
|------|------|---------|
| **ランクアップ演出** | ランク昇格時の華やかなアニメーション演出 | ⭐ MVP採用 |
| スタンプ獲得演出 | スタンプ付与時のエフェクト | 後回し（簡易版のみ） |
| ゲーミフィケーション | バッジ・実績・チャレンジ機能 | 後回し |
| AIレコメンド | 購買傾向に基づく商品提案 | 後回し |
| SNSシェア | ランクアップをSNSでシェア | 後回し |

---

## MVP機能サマリー

### 採用機能

| 品質タイプ | 機能 |
|------------|------|
| 当たり前品質 | スタンプ数表示、スタンプ獲得、会員QRコード、ランク表示、特典一覧 |
| 魅力的品質（1つ） | **ランクアップ演出** - 昇格時の華やかなアニメーションでユーザーに達成感を提供 |

### 見送り機能

| 機能 | 理由 |
|------|------|
| スタンプ獲得履歴 | 当たり前品質ではなく、MVPでは画面数を削減 |
| プッシュ通知 | バックエンド連携が必要でMVPスコープ外 |
| 店舗情報・地図 | 店舗マスタ連携が必要でMVPスコープ外 |
| ゲーミフィケーション | 複雑な機能でMVP後に検討 |
| SNSシェア | コア体験に直接関係しない |

---

## 魅力的品質の詳細: ランクアップ演出

### 選定理由

1. **達成感の最大化** - スタンプを貯める努力が報われる瞬間を演出
2. **再来店動機の強化** - 次のランクを目指したくなる心理効果
3. **デモ映え** - 営業時のデモで視覚的インパクトを与えられる
4. **実装コスト** - Framer Motionで比較的低コストで実現可能

### 演出イメージ

```
┌─────────────────────────┐
│   ★ RANK UP! ★        │
│                         │
│   ┌───────────────┐     │
│   │   GOLD        │     │
│   │   MEMBER      │     │
│   └───────────────┘     │
│   ✨ 紙吹雪エフェクト ✨  │
│                         │
│   特典: 10%OFFクーポン   │
│   獲得しました！        │
│                         │
│   [OK]                  │
└─────────────────────────┘
```

### 実装要素

- 背景のグラデーション変化
- ランクバッジのスケールイン + グロー効果
- 紙吹雪（confetti）エフェクト
- テキストのフェードイン
- バイブレーション（モバイル）
